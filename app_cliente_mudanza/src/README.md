# Estructura del Proyecto

## Carpetas

- **components/** - Componentes reutilizables
  - **common/** - Componentes comunes (Headers, Footers, etc.)
  - **ui/** - Componentes de UI básicos (Buttons, Cards, Inputs)

- **screens/** - Pantallas de la aplicación

- **navigation/** - Configuración de navegación (Stack, Tab, Drawer)

- **services/** - Lógica de servicios externos (API, Firebase, etc.)

- **hooks/** - Custom Hooks de React

- **utils/** - Funciones utilitarias y helpers

- **constants/** - Constantes y configuraciones (colores, endpoints, etc.)

- **contexts/** - Context API para estado global

- **store/** - Gestión de estado (Redux, Zustand, MobX)

- **assets/** - Recursos locales al src (si es necesario)

- **types/** - TypeScript types/interfaces

- **theme/** - Tema de la app (colores, tipografía, espaciado)
